init(integerLiteral:)

Creates a percentmille value using the given integer literal as the numerator.

Implements requirement in ExpressibleByIntegerLiteral.

Declaration

init(integerLiteral:Int)

Parameters

integerLiteral

The integer literal.

Overview

The provided integer literal is not the numerical value of the created percentmille value. It will be interpreted as the numerator of a rational value.
