Initializer

init(name:)

Creates a chunk type identifier.

Declaration

init(name:UInt32)

Parameters

name

The chunk type code. Bit 13 must be set. If the type code is not a public PNG chunk type code, then bit 29 must be clear.

Overview

The chunk type code is a four byte integer, where bit 5 in each constituent byte is a flag bit.

The flag bit in the uppermost byte (bit 29) is the ancillary bit.

The flag bit in the third byte (bit 21) is the private bit.

The flag bit in the second byte (bit 13) is reserved and must be set.

The flag bit in the lowest byte (bit 5) is the safe-to-copy bit.

Passing an invalid type code will result in a precondition failure. For a failable version of this initializer, use init(validating:). For more details on type code semantics, consult the PNG specification.
