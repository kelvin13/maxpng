Enumeration Case

indexed4

Pixels are stored as 4-bit indices.

Declaration

case indexed4

Overview

An image with this pixel format has a bit depth of 4, and a color depth of 8. Each index is in the range 0 ... 15, and can reference an entry in a palette with at most 16 elements.
