Enumeration Case

v8(fill:key:)

An 8-bit grayscale color format.

Declaration

case v8(fill:UInt8?, key:UInt8?)

Associated values

fill

An optional background color. Most PNG viewers ignore this field.

key

An optional chroma key. If present, pixels matching it will be displayed as transparent, if possible.

Overview

This color format has a pixel format of v8.
