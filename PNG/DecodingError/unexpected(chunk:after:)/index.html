Enumeration Case

unexpected(chunk:after:)

The decoder encountered a chunk of a type that is not allowed to appear after a previously encountered chunk of a particular type.

Declaration

case unexpected(chunk:Chunk, after:Chunk)

Associated values

chunk

The type of the encountered chunk.

after

The type of the preceeding chunk that precludes the encountered chunk.

Overview

If both fields are set to IDAT, this indicates a non-contiguous IDAT sequence.
