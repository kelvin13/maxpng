Instance Property

compressed

Indicates if the text is (or is to be) stored in compressed or uncompressed form within a PNG file.

Declaration

let compressed:Bool

Overview

This flag is true if the original text chunk was a zTXt chunk, and false if it was a tEXt chunk. If the original chunk was an iTXt chunk, this flag can be either true or false.
