Enumeration Case

v(_:)

A background descriptor for a grayscale or grayscale-alpha image.

Declaration

case v(UInt16)

Associated values

_

A background color.

Note that the background value is an unscaled sample. If the image color depth is less than 16, only the least-significant bits are inhabited.
