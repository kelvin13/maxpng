Generic Static Method

premultiply(_:alpha:)

Premultiplies a color component with an alpha value.

Declaration

@inlinable static func premultiply<T>(_ color:T, alpha:T) -> T where T:FixedWidthInteger & UnsignedInteger

Parameters

color

The color component to premultiply.

alpha

The alpha component to premultiply color with.

Return value

The premultiplied color component, rounded to the nearest integer.

Overview

The color and alpha parameters are interpreted as rational numbers in the range [0, 1], where T.min maps to 0, and T.max maps to 1.

This function uses no floating point operations, and satisfies the property that

premultiply(_:alpha:)

is equivalent to

premultiply(_:alpha:) ∘ straighten(_:alpha:) ∘ premultiply(_:alpha:)

The computed properties RGBA.premultiplied and VA.premultiplied can be used to premultiply an entire instance of one of the built-in color targets.

Premultiplication is a destructive operation. In the most extreme case, if alpha is T.min, this function will return T.min for any value of color.
