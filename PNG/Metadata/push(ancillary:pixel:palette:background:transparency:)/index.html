Instance Method

push(ancillary:pixel:palette:background:transparency:)

Parses an ancillary chunk, and either adds it to this metadata instance, or stores it in one of the two inout parameters.

Declaration

mutating func push(ancillary chunk:(type:Chunk, data:[UInt8]), pixel:Format.Pixel, palette:Palette?, background:inout Background?, transparency:inout Transparency?) throws

Parameters

chunk

The chunk to process.

The type identifier of this chunk must not be a critical chunk type. (Critical chunk types are CgBI, IHDR, PLTE, IDAT, and IEND.) Passing a critical chunk to this function will result in a precondition failure.

pixel

The image pixel format.

palette

The image palette, if available. Client applications are expected to set this parameter to nil if the PLTE chunk has not yet been encountered.

background

The background descriptor, if available. If this function receives a bKGD chunk, it will be parsed and stored in this variable. Client applications are expected to initialize it to nil, and should not overwrite it between subsequent calls while processing the same image.

transparency

The transparency descriptor, if available. If this function receives a tRNS chunk, it will be parsed and stored in this variable. Client applications are expected to initialize it to nil, and should not overwrite it between subsequent calls while processing the same image.

Overview

If the given chunk is a bKGD or tRNS chunk, it will be stored in its respective inout variable. Otherwise it will be stored within this metadata instance.

This function parses and validates the given chunk according to the image pixel format and palette. It also validates its multiplicity, and its chunk ordering with respect to the PLTE chunk.
